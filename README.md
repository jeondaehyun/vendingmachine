# Vending Machine


## 이름: 앨범 자판기

### 사용언어 : html,css,javascript,jquery

#### 기능 
- 앨범 금액에 맞는 돈이 있을때 해당 앨범에 가격부분이 노란색으로 변함
- 모든 구매가 완료되었을때 금액을 반환할 수 있고 그 반환하는 금액의 단위 및 수량을 볼수 있음
